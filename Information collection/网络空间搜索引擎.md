# 网络空间搜索引擎概述

网络空间搜索引擎是一种通过扫描和分析互联网资产，帮助用户快速定位目标的工具，广泛应用于网络安全、渗透测试和资产管理等领域。

## 主流网络空间搜索引擎及特点

### 1. FOFA
**FOFA** 是一款功能强大的网络空间搜索引擎，支持通过 IP 地址、端口、协议、关键字等多种方式进行搜索。它能够识别主流软硬件产品，并提供详细的资产信息，如端口、服务和设备类型等。FOFA 还支持高级语法查询，适合进行漏洞分析和资产监控。

### 2. Shodan
**Shodan** 是全球知名的网络空间搜索引擎，专注于搜索与互联网连接的设备，如路由器、摄像头、数据库等。它支持通过地理位置、端口、服务类型等条件进行搜索，并提供强大的 API 接口，适合企业进行大规模资产监控。

### 3. ZoomEye（钟馗之眼）
**ZoomEye** 是中国开发的网络空间搜索引擎，支持对全球 IPv4 和 IPv6 地址进行扫描。它提供设备指纹和 Web 指纹检索，能够识别操作系统、服务版本、地理位置等信息，同时支持可视化资产地图和趋势分析。

### 4. 360 Quake
**Quake** 是 360 公司推出的网络空间测绘系统，结合人工智能和机器学习技术，能够精准识别全球网络资产。它还整合了漏洞数据，支持对漏洞影响范围的评估，是漏洞管理和资产分类的优秀工具。

### 5. Censys
**Censys** 由密歇根大学开发，专注于扫描和分析全球互联网设备。它每天扫描 IPv4 地址空间，提供详细的设备配置和部署信息，适合安全研究人员进行资产监控和风险评估。

### 6. 奇安信 Hunter（鹰图）
**Hunter** 是奇安信推出的网络空间测绘平台，支持资产探测、端口扫描和协议解析。它通过将地理空间与网络空间映射，帮助用户快速定位互联网资产，并提供备案信息关联功能，适合国内资产管理和行业评估。
